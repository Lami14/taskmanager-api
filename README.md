# Smart Task Manager API

Backend for Smart Task Manager SaaS app.

## Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)
- JWT Authentication
- Bcrypt for password hashing

## Features
- User registration/login
- JWT authentication & protected routes
- Task CRUD
- Role-based access (admin/user)

## Setup
```bash
npm install
cp .env.example .env
npm run dev
